~/projects/autonomous-robot in progress

Autonomous Mobile Robot System

Full-stack autonomous mobile robot built from scratch, leading a 3-member team. The system implements complete ROS2 navigation — SLAM Toolbox for real-time 2D mapping, Nav2 for global and local path planning, and multi-sensor fusion combining LiDAR and Kinect RGB-D data. Distributed control runs over a Zenoh router with Tailscale mesh VPN, enabling remote monitoring and teleoperation.
